How to Make Pong In Unreal Engine

Let's make PONG!! In this part one video, I will show you how to create and set up the player, walls, and ball for pong using Unreal Engine 4. This is a classic game that is great for beginners to learn how to use Unreal Blueprints while also getting used to the editor. The game pong is simple to make and is not very taxing for your computer to handle. You will be able to make this with a minimum wait time so you can further your game development learning. In this video, we break down the Gamemode, Pawn, and Physics inside Unreal Engine. Instead of making a super long video, I broke it down into two segments. This way you will be able to keep up and can reference the video if any problems arise.
